Facebook Comments Prediction

Python, Flask, Numpy, Jupyter Notebook

Since several years, social networking services use has highly increased. This dataset fit in the need to understand the behavior of users on those services. In 2016, Facebook was the most active social network with 1.76 billion users. Thus, the goal is to predict the number of comments in the next H hours (H being a feature of the dataset).

It's in 2 parts :
* One notebook containing the data exploration, as well as the training of the models (in the API folder)
* An website version, with an API

Everything about this project is stored in this repo.
